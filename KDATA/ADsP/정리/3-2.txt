# 통계학 개론

모수 = 통계량


### 확률적 표본추출법의 종류

- 단순 무작위 추출 Simple random sampling : 표본 선택 확률 동일 n/N의 확률
- 계통 추출 (Systemetic) : dlffusqjsgh qndu gn dlfwjdrksrurdmf tjsxor
- 층화 추출 (Stratified) : 겹치지 않는 층으로 나누고 집단 내에서 특정크기의 표본을 단순 무작위 추출 (성별, 나이대, 지역)
- 군집 추출(Clustering) : 차이가 없는 여러 집단으로 나누고 몇개를 선택하고 필요한 만큼의 표본을 임의로 선택 (1반, 2반 중 몇 개를 추출)

### 오차
- 표본 (추출) 오차 
	- 모집단을 대표하지 못하는 표본을 추출하여 발생하는 오차
	- 표본의 크기가 커지면 작아 짐, 전수조사시 0
- 비표본 추출 오차
	- 표본 오차를 제외한 모든 오차
	- 표본 크기에 비례 
- 편의(bias)
	- 확률화(randomize)로 없애거나 최소화 가능

### 데이터 특성
- 서열척도 : 순위, 양적 비교 불가
- 등간척도(구간척도) : 순위가 있으나 덧셈, 뺄셈 가능 (온도계, 물가지수)
- 비율척도 : 절대 0점 존재, 사칙연산 가능 (몸무게, 나이)

### 집중화 경향
첨도, 왜도, 집중화 경향(mean, median, mode 비교) 확인

### 데이터 퍼짐 정도
산포도, 범위, IQR, 편차 간단한 개념

### 표본분산, 표본 표준편차
- 표본 분산 : 평균과 비슷하지만 다를 수 있음
편차의 제곱의 합을 n-1로 나눔
- 표본 표준편차
자료의 산포도를 나타내는 수치, 분산의 양의 제곱근
